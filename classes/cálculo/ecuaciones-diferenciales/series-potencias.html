Series de Potencias

Ecuaciones Diferenciales — Soluciones analíticas mediante series

Introducción

Muchas EDO importantes no tienen solución en términos de funciones elementales. El método de series de potencias permite encontrar soluciones como sumas infinitas, que convergen en un intervalo alrededor de un punto. Este método es indispensable en física matemática: las funciones de Bessel, Legendre, Hermite y Laguerre se obtienen así.

Serie de potencias alrededor de \(x_0\): \[ y(x) = \sum_{n=0}^{\infty} a_n (x - x_0)^n = a_0 + a_1(x-x_0) + a_2(x-x_0)^2 + \cdots \] El radio de convergencia \(R\) se determina mediante: \[ R = \frac{1}{\limsup_{n\to\infty} |a_n|^{1/n}} \quad\text{o}\quad R = \lim_{n\to\infty}\left|\frac{a_n}{a_{n+1}}\right| \]

La idea central es sustituir la serie en la EDO, igualar coeficientes de cada potencia de \((x-x_0)^n\) a cero, y obtener una relación de recurrencia para los coeficientes \(a_n\).

Método general

Procedimiento: Para resolver \(y'' + P(x)y' + Q(x)y = 0\) alrededor de un punto ordinario \(x_0\):
  1. Proponer \(y = \sum_{n=0}^{\infty} a_n(x-x_0)^n\).
  2. Calcular \(y' = \sum_{n=1}^{\infty} na_n(x-x_0)^{n-1}\) e \(y'' = \sum_{n=2}^{\infty} n(n-1)a_n(x-x_0)^{n-2}\).
  3. Sustituir en la EDO y reindexar para que todas las sumas tengan la misma potencia.
  4. Igualar a cero el coeficiente de cada potencia \((x-x_0)^k\).
  5. Obtener la relación de recurrencia para \(a_n\).
  6. Los coeficientes \(a_0\) y \(a_1\) quedan libres (dos constantes arbitrarias).
Punto ordinario vs. singular: Un punto \(x_0\) es ordinario si \(P(x)\) y \(Q(x)\) son analíticas en \(x_0\). Si alguna tiene un polo, \(x_0\) es un punto singular. El radio de convergencia mínimo es la distancia al punto singular más cercano (incluso en el plano complejo).

Puntos singulares

Para la ecuación \(y'' + P(x)y' + Q(x)y = 0\), un punto \(x_0\) donde \(P\) o \(Q\) no son analíticas es singular:

TipoCondiciónMétodo
Punto ordinario \(P(x_0)\) y \(Q(x_0)\) analíticas Series de potencias ordinarias
Singular regular \((x-x_0)P(x)\) y \((x-x_0)^2Q(x)\) analíticas Método de Frobenius
Singular irregular No cumple las anteriores Métodos asintóticos / WKB

Método de Frobenius

Cuando \(x_0\) es un punto singular regular, el método de Frobenius generaliza la serie de potencias incluyendo un exponente fraccionario:

Serie de Frobenius: \[ y = (x-x_0)^r \sum_{n=0}^{\infty} a_n (x-x_0)^n = \sum_{n=0}^{\infty} a_n (x-x_0)^{n+r} \] donde \(r\) se determina resolviendo la ecuación indicial.
Ecuación indicial: Al sustituir la serie de Frobenius en la EDO, el coeficiente de la potencia más baja da: \[ r(r-1) + p_0\,r + q_0 = 0 \] donde \(p_0 = \lim_{x\to x_0}(x-x_0)P(x)\) y \(q_0 = \lim_{x\to x_0}(x-x_0)^2Q(x)\).

Las dos raíces \(r_1, r_2\) de la ecuación indicial determinan tres casos:

  • Si \(r_1 - r_2 \notin \mathbb{Z}\): dos series de Frobenius independientes.
  • Si \(r_1 = r_2\): la segunda solución incluye un término logarítmico.
  • Si \(r_1 - r_2 \in \mathbb{Z}^+\): puede requerirse un término logarítmico.

Ejemplos resueltos

Ejemplo 1: Ecuación de Airy

Resolver \(y'' = xy\) mediante series de potencias alrededor de \(x_0 = 0\).

Proponemos \(y = \sum_{n=0}^{\infty} a_n x^n\). Entonces \(y'' = \sum_{n=2}^{\infty} n(n-1)a_n x^{n-2}\).

Sustituyendo:

\[ \sum_{n=2}^{\infty} n(n-1)a_n x^{n-2} = x\sum_{n=0}^{\infty} a_n x^n = \sum_{n=0}^{\infty} a_n x^{n+1} \]

Reindexamos el lado izquierdo con \(k = n-2\): \(\sum_{k=0}^{\infty}(k+2)(k+1)a_{k+2}x^k\).

Lado derecho con \(k = n+1\): \(\sum_{k=1}^{\infty}a_{k-1}x^k\).

Para \(k = 0\): \(2a_2 = 0 \;\Rightarrow\; a_2 = 0\).

Para \(k \geq 1\): relación de recurrencia:

\[ a_{k+2} = \frac{a_{k-1}}{(k+2)(k+1)}, \quad k \geq 1 \]

Los primeros términos:

\[ y = a_0\!\left(1 + \frac{x^3}{6} + \frac{x^6}{180} + \cdots\right) + a_1\!\left(x + \frac{x^4}{12} + \frac{x^7}{504} + \cdots\right) \]

La primera serie es proporcional a \(\text{Ai}(x)\) y la segunda a \(\text{Bi}(x)\).

Ejemplo 2: Ecuación de Bessel de orden 0

Resolver \(x^2y'' + xy' + x^2y = 0\) (Bessel con \(\nu = 0\)) mediante Frobenius.

En forma estándar: \(y'' + \frac{1}{x}y' + y = 0\). El punto \(x = 0\) es singular regular.

Ecuación indicial: \(r^2 = 0 \;\Rightarrow\; r = 0\) (raíz doble).

Serie: \(y = \sum_{n=0}^{\infty}a_nx^n\). La recurrencia da:

\[ a_{n+2} = -\frac{a_n}{(n+2)^2} \]

Con \(a_0 = 1\), \(a_1 = 0\) (solo potencias pares):

\[ J_0(x) = \sum_{k=0}^{\infty}\frac{(-1)^k}{(k!)^2}\left(\frac{x}{2}\right)^{2k} = 1 - \frac{x^2}{4} + \frac{x^4}{64} - \frac{x^6}{2304} + \cdots \]

Ejemplo 3: Polinomios de Legendre

La ecuación de Legendre \((1-x^2)y'' - 2xy' + l(l+1)y = 0\) con \(l = 2\).

La serie de potencias da una recurrencia que termina cuando \(n = l\), produciendo un polinomio:

\[ P_2(x) = \frac{1}{2}(3x^2 - 1) \]

Los polinomios de Legendre satisfacen la ortogonalidad:

\[ \int_{-1}^{1} P_m(x)\,P_n(x)\,dx = \frac{2}{2n+1}\,\delta_{mn} \]

Son fundamentales para expandir funciones en armónicos esféricos.
