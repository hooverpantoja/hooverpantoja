EDP y Separación de Variables

Ecuaciones Diferenciales — Ecuaciones en derivadas parciales fundamentales

Introducción a las EDP

Una ecuación en derivadas parciales (EDP) relaciona una función de varias variables con sus derivadas parciales. Son el lenguaje natural de la física: describen la propagación del calor, las ondas, la electrostática, la mecánica de fluidos y la mecánica cuántica.

Forma general de una EDP lineal de segundo orden en dos variables: \[ A\frac{\partial^2 u}{\partial x^2} + 2B\frac{\partial^2 u}{\partial x\,\partial y} + C\frac{\partial^2 u}{\partial y^2} + D\frac{\partial u}{\partial x} + E\frac{\partial u}{\partial y} + Fu = G \] donde \(A, B, C, D, E, F, G\) pueden depender de \(x\) e \(y\).

Clasificación de EDP de segundo orden

La naturaleza de la EDP depende del signo del discriminante \(\Delta = B^2 - AC\):

TipoCondiciónEcuación modeloFenómeno
Elíptica \(B^2 - AC < 0\) \(\nabla^2 u = 0\) (Laplace) Estado estacionario, equilibrio
Parabólica \(B^2 - AC = 0\) \(u_t = \alpha^2 u_{xx}\) (calor) Difusión, conducción térmica
Hiperbólica \(B^2 - AC > 0\) \(u_{tt} = c^2 u_{xx}\) (onda) Ondas, vibraciones
Analogía con cónicas: La clasificación es idéntica a la de las secciones cónicas. Al igual que \(Ax^2 + 2Bxy + Cy^2\) define una elipse, parábola o hipérbola, el discriminante \(B^2 - AC\) determina la geometría de las curvas características de la EDP.

Método de separación de variables

El método de separación de variables (o método de Fourier) es la técnica más poderosa para resolver EDP lineales con condiciones de frontera. La idea es proponer que la solución se puede escribir como un producto de funciones de una sola variable:

Separación de variables: Proponer \(u(x,t) = X(x)\,T(t)\). Sustituir en la EDP para obtener: \[ \frac{X''(x)}{X(x)} = \frac{1}{\alpha^2}\frac{T'(t)}{T(t)} = -\lambda \] Cada lado depende de una variable distinta, así que ambos deben ser iguales a una constante \(-\lambda\) (la constante de separación). Esto genera dos EDO: \[ X'' + \lambda X = 0, \qquad T' + \alpha^2\lambda T = 0 \]

Pasos del método:

  1. Proponer \(u = X(x)\,T(t)\) y sustituir en la EDP.
  2. Separar variables: cada lado depende de una sola variable.
  3. Igualar ambos lados a la constante de separación \(-\lambda\).
  4. Resolver la EDO espacial \(X'' + \lambda X = 0\) con las condiciones de frontera para obtener los valores propios \(\lambda_n\) y las funciones propias \(X_n\).
  5. Resolver la EDO temporal \(T_n\).
  6. Formar la solución general como superposición: \(u = \sum_{n} c_n X_n(x)\,T_n(t)\).
  7. Determinar los coeficientes \(c_n\) con la condición inicial (series de Fourier).

Ecuación del calor

Ecuación del calor unidimensional: \[ \frac{\partial u}{\partial t} = \alpha^2 \frac{\partial^2 u}{\partial x^2}, \quad 0 < x < L, \quad t > 0 \] Condiciones de frontera (Dirichlet): \(u(0,t) = 0\), \(u(L,t) = 0\).
Condición inicial: \(u(x,0) = f(x)\).

Separando variables \(u = X(x)T(t)\):

\[ X'' + \lambda X = 0,\quad X(0)=0,\ X(L)=0 \;\Longrightarrow\; \lambda_n = \left(\frac{n\pi}{L}\right)^2,\ X_n = \sin\frac{n\pi x}{L} \] \[ T_n' + \alpha^2\lambda_n T_n = 0 \;\Longrightarrow\; T_n = e^{-\alpha^2(n\pi/L)^2 t} \]
Solución general: \[ u(x,t) = \sum_{n=1}^{\infty} b_n \sin\frac{n\pi x}{L}\,e^{-\alpha^2(n\pi/L)^2 t} \] donde los coeficientes de Fourier son: \[ b_n = \frac{2}{L}\int_0^L f(x)\sin\frac{n\pi x}{L}\,dx \]

Observa que cada modo \(n\) decae exponencialmente, con los modos de frecuencia alta decayendo mucho más rápido que los de baja frecuencia.

Ecuación de onda

Ecuación de onda unidimensional: \[ \frac{\partial^2 u}{\partial t^2} = c^2 \frac{\partial^2 u}{\partial x^2}, \quad 0 < x < L, \quad t > 0 \] Modela vibraciones de una cuerda fija en ambos extremos.

Separando variables se obtiene:

\[ u(x,t) = \sum_{n=1}^{\infty}\left(A_n\cos\frac{n\pi ct}{L} + B_n\sin\frac{n\pi ct}{L}\right)\sin\frac{n\pi x}{L} \]

Cada término \(n\) es un modo normal o armónico de la cuerda, con frecuencia \(f_n = \frac{nc}{2L}\). El primer modo (\(n=1\)) es la frecuencia fundamental.

Fórmula de d'Alembert: En un dominio infinito, la solución de la ecuación de onda tiene una forma particularmente elegante: \[ u(x,t) = \frac{1}{2}\!\left[f(x-ct) + f(x+ct)\right] + \frac{1}{2c}\int_{x-ct}^{x+ct} g(s)\,ds \] que se interpreta como dos ondas viajando en direcciones opuestas con velocidad \(c\).

Ecuación de Laplace

Ecuación de Laplace: \[ \nabla^2 u = \frac{\partial^2 u}{\partial x^2} + \frac{\partial^2 u}{\partial y^2} = 0 \] Las soluciones se llaman funciones armónicas. Describen potenciales electrostáticos, temperaturas en estado estacionario y flujos irrotacionales.

En un rectángulo \([0,a]\times[0,b]\) con \(u=0\) en tres lados y \(u(x,b)=f(x)\):

\[ u(x,y) = \sum_{n=1}^{\infty} c_n \sin\frac{n\pi x}{a}\,\frac{\sinh(n\pi y/a)}{\sinh(n\pi b/a)} \]
Principio del máximo: Una función armónica en un dominio acotado alcanza su máximo y mínimo en la frontera, nunca en el interior. Esto implica que la solución del problema de Dirichlet es única.

Ejemplos resueltos

Ejemplo 1: Barra con temperatura inicial triangular

Resolver la ecuación del calor en \([0, \pi]\) con \(\alpha = 1\), \(u(0,t) = u(\pi,t) = 0\), y condición inicial:

\[ f(x) = \begin{cases} x & 0 \leq x \leq \pi/2 \\ \pi - x & \pi/2 \leq x \leq \pi \end{cases} \]

Solución: Los coeficientes de Fourier son:

\[ b_n = \frac{2}{\pi}\int_0^{\pi} f(x)\sin(nx)\,dx = \frac{4}{\pi n^2}\sin\frac{n\pi}{2} \]

Entonces \(b_n = 0\) si \(n\) es par, y para \(n\) impar:

\[ b_1 = \frac{4}{\pi}, \quad b_3 = -\frac{4}{9\pi}, \quad b_5 = \frac{4}{25\pi}, \;\ldots \] \[ \boxed{u(x,t) = \frac{4}{\pi}\sum_{k=0}^{\infty}\frac{(-1)^k}{(2k+1)^2}\sin\!\big((2k+1)x\big)\,e^{-(2k+1)^2t}} \]

Ejemplo 2: Cuerda vibrante pulsada

Una cuerda de longitud \(L\) es desplazada una distancia \(h\) en su punto medio y liberada desde el reposo. Hallar \(u(x,t)\).

Condición inicial:

\[ u(x,0) = \begin{cases} \frac{2hx}{L} & 0 \leq x \leq L/2 \\ \frac{2h(L-x)}{L} & L/2 \leq x \leq L \end{cases}, \quad u_t(x,0) = 0 \]

Como la velocidad inicial es cero, \(B_n = 0\). Los coeficientes son:

\[ A_n = \frac{8h}{n^2\pi^2}\sin\frac{n\pi}{2} \] \[ \boxed{u(x,t) = \frac{8h}{\pi^2}\sum_{k=0}^{\infty}\frac{(-1)^k}{(2k+1)^2}\sin\frac{(2k+1)\pi x}{L}\cos\frac{(2k+1)\pi ct}{L}} \]

Ejemplo 3: Potencial estacionario en un rectángulo

Resolver \(\nabla^2 u = 0\) en \([0,1]\times[0,1]\) con \(u = 0\) en tres lados y \(u(x,1) = \sin(\pi x)\).

Usando separación de variables:

\[ u(x,y) = \sum_{n=1}^{\infty} c_n\sin(n\pi x)\frac{\sinh(n\pi y)}{\sinh(n\pi)} \]

La condición \(u(x,1) = \sin(\pi x)\) da \(c_1 = 1\) y \(c_n = 0\) para \(n \geq 2\).

\[ \boxed{u(x,y) = \sin(\pi x)\frac{\sinh(\pi y)}{\sinh(\pi)}} \]
