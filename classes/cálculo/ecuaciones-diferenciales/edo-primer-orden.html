EDO de Primer Orden

Ecuaciones Diferenciales — Métodos fundamentales de solución

Introducción

Una ecuación diferencial ordinaria (EDO) de primer orden relaciona una función desconocida \(y(x)\) con su primera derivada \(y'(x)\). La forma general es:

Forma general: \[ F\!\left(x,\, y,\, \frac{dy}{dx}\right) = 0 \] o equivalentemente, en forma explícita: \[ \frac{dy}{dx} = f(x, y) \]

Las EDO de primer orden modelan procesos donde la tasa de cambio de una cantidad depende del estado actual del sistema: crecimiento poblacional, decaimiento radiactivo, circuitos eléctricos RC, cinética química, entre muchos otros.

Notación: Usaremos indistintamente \(\frac{dy}{dx}\), \(y'\) o \(\dot{y}\) para la primera derivada. La primera es la notación de Leibniz, la segunda de Lagrange y la tercera de Newton (usada cuando la variable independiente es el tiempo).

Ecuaciones de variables separables

Una EDO es separable si puede escribirse de modo que las variables \(x\) e \(y\) queden en lados opuestos de la ecuación:

Forma separable: \[ \frac{dy}{dx} = g(x)\,h(y) \quad\Longrightarrow\quad \frac{dy}{h(y)} = g(x)\,dx \] Método de solución: Integrar ambos lados: \[ \int \frac{dy}{h(y)} = \int g(x)\,dx + C \]

Este es el método más elemental y el primero que se debe intentar. Si la ecuación se puede factorizar como producto de una función de \(x\) y otra de \(y\), es separable.

Ejemplo rápido: Resolver \(\frac{dy}{dx} = xy\).

Separamos variables: \(\frac{dy}{y} = x\,dx\)

\[ \int \frac{dy}{y} = \int x\,dx \;\Longrightarrow\; \ln|y| = \frac{x^2}{2} + C \;\Longrightarrow\; y = Ae^{x^2/2} \]

donde \(A = \pm e^C\) es una constante arbitraria.

EDO lineales de primer orden

Una EDO es lineal de primer orden cuando tiene la forma:

Forma estándar: \[ \frac{dy}{dx} + P(x)\,y = Q(x) \] Factor integrante: \[ \mu(x) = e^{\int P(x)\,dx} \] Solución general: \[ y(x) = \frac{1}{\mu(x)}\left[\int \mu(x)\,Q(x)\,dx + C\right] \]

El truco consiste en multiplicar toda la ecuación por el factor integrante \(\mu(x)\), lo que convierte el lado izquierdo en la derivada de un producto:

\[ \frac{d}{dx}\!\left[\mu(x)\,y\right] = \mu(x)\,Q(x) \]
¿Por qué funciona? Al multiplicar por \(\mu(x) = e^{\int P\,dx}\), se logra que \(\mu' = P\mu\). Entonces: \[ \mu\,y' + \mu\,P\,y = \mu\,y' + \mu'\,y = \frac{d}{dx}(\mu\,y) \] Es decir, la regla del producto opera "al revés" para deshacer la ecuación.

Ejemplo: Resolver \(y' + 2y = e^{-x}\).

\(P(x) = 2\), \(Q(x) = e^{-x}\).

Factor integrante: \(\mu = e^{\int 2\,dx} = e^{2x}\).

\[ \frac{d}{dx}(e^{2x}y) = e^{2x}\cdot e^{-x} = e^{x} \] \[ e^{2x}y = \int e^{x}\,dx = e^{x} + C \] \[ y = e^{-x} + Ce^{-2x} \]

Ecuaciones exactas

Una ecuación de la forma \(M(x,y)\,dx + N(x,y)\,dy = 0\) es exacta si existe una función \(\Phi(x,y)\) tal que:

Condición de exactitud: \[ \frac{\partial M}{\partial y} = \frac{\partial N}{\partial x} \] Si se cumple, existe \(\Phi(x,y)\) con \(\frac{\partial \Phi}{\partial x} = M\) y \(\frac{\partial \Phi}{\partial y} = N\), y la solución es: \[ \Phi(x,y) = C \]

Procedimiento:

  1. Verificar que \(\frac{\partial M}{\partial y} = \frac{\partial N}{\partial x}\).
  2. Integrar \(\Phi = \int M\,dx + g(y)\) (tratando \(y\) como constante).
  3. Determinar \(g(y)\) usando la condición \(\frac{\partial \Phi}{\partial y} = N\).
  4. La solución implícita es \(\Phi(x,y) = C\).

Ejemplo: Resolver \((2xy + 3)\,dx + (x^2 + 4y)\,dy = 0\).

Verificamos: \(M = 2xy + 3\), \(N = x^2 + 4y\).

\[ \frac{\partial M}{\partial y} = 2x = \frac{\partial N}{\partial x} \quad \checkmark \]

Integramos: \(\Phi = \int (2xy+3)\,dx = x^2y + 3x + g(y)\).

Derivamos: \(\frac{\partial \Phi}{\partial y} = x^2 + g'(y) = x^2 + 4y \;\Rightarrow\; g'(y) = 4y \;\Rightarrow\; g(y) = 2y^2\).

Solución: \(x^2y + 3x + 2y^2 = C\).

Ecuación de Bernoulli

La ecuación de Bernoulli es una EDO no lineal que se puede reducir a una lineal mediante sustitución:

Forma de Bernoulli: \[ \frac{dy}{dx} + P(x)\,y = Q(x)\,y^n \quad (n \neq 0,\,1) \] Sustitución: \(v = y^{1-n}\), con lo que: \[ \frac{dv}{dx} + (1-n)\,P(x)\,v = (1-n)\,Q(x) \] que es lineal en \(v\).

Ejemplo: Resolver \(y' + y = y^2\).

Aquí \(n = 2\), así que \(v = y^{-1}\), \(v' = -y^{-2}y'\).

Dividimos por \(y^2\): \(y^{-2}y' + y^{-1} = 1\), es decir \(-v' + v = 1\), o bien:

\[ v' - v = -1 \]

Factor integrante: \(\mu = e^{-x}\). Entonces \(\frac{d}{dx}(ve^{-x}) = -e^{-x}\).

\[ ve^{-x} = e^{-x} + C \;\Rightarrow\; v = 1 + Ce^{x} \;\Rightarrow\; y = \frac{1}{1 + Ce^{x}} \]
Ecuación logística: La ecuación de Bernoulli con \(n = 2\) genera el modelo logístico de crecimiento poblacional: \(\frac{dP}{dt} = rP(1 - P/K)\), donde \(r\) es la tasa de crecimiento y \(K\) la capacidad de carga.

Teorema de existencia y unicidad

Teorema de Picard-Lindelöf: Dada la EDO con condición inicial: \[ y' = f(x,y), \quad y(x_0) = y_0 \] Si \(f\) y \(\frac{\partial f}{\partial y}\) son continuas en un rectángulo alrededor de \((x_0, y_0)\), entonces existe una única solución local \(y(x)\) definida en un intervalo \((x_0 - \delta,\, x_0 + \delta)\).

Este teorema garantiza que, bajo condiciones suaves, las curvas solución nunca se cruzan. Es fundamental para la interpretación geométrica de los campos de direcciones: las flechas del campo definen trayectorias únicas que no se intersecan.

Contraejemplo clásico: La EDO \(y' = 3y^{2/3}\) con \(y(0) = 0\) viola la condición de Lipschitz en \(y = 0\) (pues \(\frac{\partial f}{\partial y} = 2y^{-1/3}\) diverge). Tiene infinitas soluciones, incluyendo \(y = 0\) y \(y = (x - c)^3\) para cualquier \(c \geq 0\).

Ejemplos resueltos

Ejemplo 1: Decaimiento radiactivo

Un material radiactivo decae a una tasa proporcional a su masa actual. Si inicialmente hay 100 g y tras 5 años quedan 80 g, hallar \(m(t)\).

Modelo: \(\frac{dm}{dt} = -\lambda m\), \(m(0) = 100\).

Solución: Es separable:

\[ \int \frac{dm}{m} = -\lambda\int dt \;\Longrightarrow\; m(t) = 100\,e^{-\lambda t} \]

Aplicando \(m(5) = 80\):

\[ 80 = 100\,e^{-5\lambda} \;\Longrightarrow\; \lambda = -\frac{1}{5}\ln\!\left(\frac{4}{5}\right) = \frac{\ln 5 - \ln 4}{5} \approx 0.04463 \] \[ \boxed{m(t) = 100\,e^{-0.04463\,t} \text{ gramos}} \]

Ejemplo 2: Circuito RC

En un circuito RC con fuente de voltaje \(V_0\), la carga \(q(t)\) satisface:

\[ R\frac{dq}{dt} + \frac{q}{C} = V_0, \quad q(0) = 0 \]

Solución: Reescribimos en forma estándar: \(q' + \frac{1}{RC}q = \frac{V_0}{R}\).

Factor integrante: \(\mu = e^{t/(RC)}\).

\[ \frac{d}{dt}\!\left(q\,e^{t/(RC)}\right) = \frac{V_0}{R}\,e^{t/(RC)} \] \[ q\,e^{t/(RC)} = V_0C\,e^{t/(RC)} + K \]

Con \(q(0) = 0\): \(K = -V_0C\).

\[ \boxed{q(t) = V_0C\!\left(1 - e^{-t/(RC)}\right)} \]

La constante de tiempo \(\tau = RC\) determina la rapidez de carga del condensador.

Ejemplo 3: Ecuación exacta

Resolver \((ye^{xy} + 2x)\,dx + (xe^{xy} + 2y)\,dy = 0\).

\(M = ye^{xy} + 2x\), \(N = xe^{xy} + 2y\).

Verificamos exactitud:

\[ \frac{\partial M}{\partial y} = e^{xy} + xye^{xy}, \qquad \frac{\partial N}{\partial x} = e^{xy} + xye^{xy} \quad \checkmark \]

Buscamos \(\Phi\):

\[ \Phi = \int (ye^{xy}+2x)\,dx = e^{xy} + x^2 + g(y) \] \[ \frac{\partial \Phi}{\partial y} = xe^{xy} + g'(y) = xe^{xy} + 2y \;\Rightarrow\; g'(y) = 2y \;\Rightarrow\; g(y) = y^2 \]

Solución: \(\boxed{e^{xy} + x^2 + y^2 = C}\)
