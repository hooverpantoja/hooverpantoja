EDO de Segundo Orden

Ecuaciones Diferenciales — Coeficientes constantes, resonancia y oscilaciones

Introducción

Las EDO lineales de segundo orden son las más importantes en la física e ingeniería. Modelan sistemas oscilatorios, vibraciones, circuitos RLC y la mecánica cuántica.

Forma general: \[ a\,\frac{d^2y}{dx^2} + b\,\frac{dy}{dx} + c\,y = g(x) \] donde \(a, b, c\) son constantes (o funciones de \(x\)) y \(g(x)\) es el término forzante.
  • Si \(g(x) = 0\): ecuación homogénea.
  • Si \(g(x) \neq 0\): ecuación no homogénea.
Principio de superposición: Si \(y_1\) e \(y_2\) son soluciones de la ecuación homogénea, entonces \(y = c_1y_1 + c_2y_2\) también lo es. La solución general de la no homogénea es \(y = y_h + y_p\), donde \(y_h\) es la solución homogénea y \(y_p\) una solución particular.

EDO homogéneas con coeficientes constantes

Para \(ay'' + by' + cy = 0\), proponemos \(y = e^{rx}\). Sustituyendo:

\[ ar^2e^{rx} + bre^{rx} + ce^{rx} = 0 \;\Longrightarrow\; e^{rx}(ar^2 + br + c) = 0 \]
Ecuación característica: \[ ar^2 + br + c = 0 \] Las raíces \(r_1, r_2\) determinan la forma de la solución: \[ r = \frac{-b \pm \sqrt{b^2 - 4ac}}{2a} \] El discriminante \(\Delta = b^2 - 4ac\) clasifica los tres casos posibles.

Clasificación según las raíces

DiscriminanteRaícesSolución generalComportamiento
\(\Delta > 0\) Reales distintas \(r_1 \neq r_2\) \(y = c_1e^{r_1x} + c_2e^{r_2x}\) Exponencial (sobreamortiguado)
\(\Delta = 0\) Real repetida \(r_1 = r_2 = r\) \(y = (c_1 + c_2x)e^{rx}\) Críticamente amortiguado
\(\Delta < 0\) Complejas \(r = \alpha \pm \beta i\) \(y = e^{\alpha x}(c_1\cos\beta x + c_2\sin\beta x)\) Oscilatorio (subamortiguado)
Caso oscilatorio: Si \(\alpha < 0\), las oscilaciones se amortiguan. Si \(\alpha = 0\), se obtiene movimiento armónico simple con frecuencia \(\omega = \beta\). Si \(\alpha > 0\), las oscilaciones crecen sin límite (inestabilidad).

Ecuaciones no homogéneas: coeficientes indeterminados

Para \(ay'' + by' + cy = g(x)\), la solución general es \(y = y_h + y_p\). El método de coeficientes indeterminados propone una forma para \(y_p\) según \(g(x)\):

Término forzante \(g(x)\)Propuesta \(y_p\)
\(P_n(x)\) (polinomio grado \(n\)) \(x^s(A_nx^n + \cdots + A_1x + A_0)\)
\(e^{\alpha x}\) \(x^s A e^{\alpha x}\)
\(\cos\beta x\) o \(\sin\beta x\) \(x^s(A\cos\beta x + B\sin\beta x)\)
\(e^{\alpha x}\cos\beta x\) \(x^s e^{\alpha x}(A\cos\beta x + B\sin\beta x)\)

Donde \(s\) es el número de veces que la propuesta coincide con una solución de la homogénea (regla de modificación).

Ejemplo: Resolver \(y'' + 4y = \cos 2x\).

Homogénea: \(r^2 + 4 = 0 \;\Rightarrow\; r = \pm 2i\). Entonces \(y_h = c_1\cos 2x + c_2\sin 2x\).

Como \(\cos 2x\) ya es solución homogénea, proponemos \(y_p = x(A\cos 2x + B\sin 2x)\).

Derivando y sustituyendo (tras simplificar):

\[ -4A\sin 2x + 4B\cos 2x = \cos 2x \] \[ A = 0, \quad B = \frac{1}{4} \] \[ \boxed{y = c_1\cos 2x + c_2\sin 2x + \frac{x}{4}\sin 2x} \]

El último término crece linealmente: es resonancia.

Variación de parámetros

Método general que funciona para cualquier \(g(x)\), no solo exponenciales y trigonométricas:

Método de variación de parámetros: Dadas \(y_1, y_2\) soluciones de la homogénea, la solución particular es: \[ y_p = -y_1\int\frac{y_2\,g(x)}{aW}\,dx + y_2\int\frac{y_1\,g(x)}{aW}\,dx \] donde \(W = W(y_1, y_2)\) es el Wronskiano.

El Wronskiano

Wronskiano: Para dos funciones \(y_1, y_2\): \[ W(y_1,y_2) = \begin{vmatrix} y_1 & y_2 \\ y_1' & y_2' \end{vmatrix} = y_1y_2' - y_2y_1' \]
  • \(W \neq 0\): las soluciones son linealmente independientes y forman un sistema fundamental.
  • \(W = 0\): las soluciones son linealmente dependientes (una es múltiplo de la otra).

Ejemplo: Verificar que \(y_1 = e^x\) e \(y_2 = e^{-x}\) son linealmente independientes.

\[ W = \begin{vmatrix} e^x & e^{-x} \\ e^x & -e^{-x} \end{vmatrix} = -e^x\cdot e^{-x} - e^x\cdot e^{-x} = -1 - 1 = -2 \neq 0 \quad \checkmark \]

Ejemplos resueltos

Ejemplo 1: Oscilador armónico amortiguado

Resolver \(y'' + 2y' + 5y = 0\), con \(y(0) = 1\), \(y'(0) = 0\).

Ecuación característica: \(r^2 + 2r + 5 = 0\).

\[ r = \frac{-2 \pm \sqrt{4 - 20}}{2} = \frac{-2 \pm 4i}{2} = -1 \pm 2i \]

Solución general: \(y = e^{-x}(c_1\cos 2x + c_2\sin 2x)\).

Aplicando condiciones iniciales:

  • \(y(0) = c_1 = 1\)
  • \(y'(0) = -c_1 + 2c_2 = 0 \;\Rightarrow\; c_2 = \frac{1}{2}\)
\[ \boxed{y = e^{-x}\!\left(\cos 2x + \frac{1}{2}\sin 2x\right)} \]

Oscilaciones de frecuencia \(\omega = 2\) que decaen exponencialmente con tasa \(\alpha = 1\).

Ejemplo 2: Masa-resorte forzado

Resolver \(y'' + 9y = 6\cos 3x\) (resonancia).

Homogénea: \(r^2 + 9 = 0 \;\Rightarrow\; r = \pm 3i\). Entonces \(y_h = c_1\cos 3x + c_2\sin 3x\).

Como \(\cos 3x\) es solución homogénea, proponemos \(y_p = x(A\cos 3x + B\sin 3x)\).

Sustituyendo y simplificando: \(-6A\sin 3x + 6B\cos 3x = 6\cos 3x\).

\[ A = 0, \quad B = 1 \] \[ \boxed{y = c_1\cos 3x + c_2\sin 3x + x\sin 3x} \]

El término \(x\sin 3x\) representa resonancia pura: la amplitud crece linealmente sin límite.

Ejemplo 3: Variación de parámetros

Resolver \(y'' + y = \sec x\).

Homogénea: \(y_h = c_1\cos x + c_2\sin x\).

Wronskiano: \(W = \cos x\cdot\cos x - \sin x\cdot(-\sin x) = 1\).

Variación de parámetros:

\[ y_p = -\cos x\int \sin x\cdot\sec x\,dx + \sin x\int \cos x\cdot\sec x\,dx \] \[ y_p = -\cos x\int \tan x\,dx + \sin x\int dx \] \[ y_p = -\cos x\cdot(-\ln|\cos x|) + \sin x\cdot x \] \[ \boxed{y_p = \cos x\,\ln|\cos x| + x\sin x} \]
